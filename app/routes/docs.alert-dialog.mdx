## Anatomy

Import the component and assemble its parts:

```tsx filename="Anatomy"
<AlertDialog>
  <AlertDialogTrigger />
  <AlertDialogContent>
    <AlertDialogHeader>
      <AlertDialogTitle />
    </AlertDialogHeader>
    <AlertDialogBody>
      <AlertDialogDescription />
    </AlertDialogBody>
    <AlertDialogFooter>
      <AlertDialogClose />
    </AlertDialogFooter>
  </AlertDialogContent>
</AlertDialog>
```

## Usage

### Trigger

You can pass a custom trigger element to the `AlertDialogTrigger` component
by using the `render` prop.

### Confirmation

You may want to alert the user before closing the dialog. 
This can be achieved by nesting the `AlertDialog` component inside the `Dialog` component.

Try to input some feedback and then click the "Cancel" button. You will be prompted with a confirmation dialog.
