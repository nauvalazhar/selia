## Anatomy

Import the component and assemble its parts:

```tsx filename="Anatomy"
<Divider>
  <DividerText />
</Divider>
```

## Usage

### Text

You can add text to the divider by using the `DividerText` component.
Use the `variant` prop to change the position of the text.
