
# Badge

Small labels to highlight status, counts, or categories on elements.

<Preview name="basic" />
## Anatomy

Import the component and assemble its parts:

```tsx filename="Anatomy"
<Badge />
```

## Usage

### Variants

Badges come in a variety of variants, each with its own color and style.

<Preview name="variants" />
### Subtle Colors

You may want to use subtle badges to create a more subtle appearance.
This can be achieved by using the `variant="*-subtle"` prop.

<Preview name="subtle" />
### Size

The `size` prop can be used to change the size of the badge.

<Preview name="size" />
### Pill

The `pill` prop can be used to create a pill-shaped badge.

<Preview name="pill" />
### Icon

You can add an icon to the badge.
