# Autocomplete

An input that suggests options as you type.

<Preview name="basic" />

## Anatomy

Import the component and assemble its parts:

```tsx filename="Anatomy"
<Autocomplete>
  <AutocompleteInput />
  <AutocompleteTrigger />
  <AutocompleteIcon />
  <AutocompleteClear />
  <AutocompleteValue />
  <AutocompleteContent>
    <AutocompleteEmpty />
    <AutocompleteList>
      <AutocompleteRow>
        <AutocompleteItem />
      </AutocompleteRow>
      <AutocompleteSeparator />
      <AutocompleteGroup>
        <AutocompleteGroupLabel />
      </AutocompleteGroup>
      <AutocompleteCollection />
    </AutocompleteList>
  </AutocompleteContent>
</Autocomplete>
```
