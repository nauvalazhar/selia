## Usage

### Import

Import the component and its related parts.

```ts filename="Button Parts"
import { Button } from "components/selia/button";
```

### Structure

The expected structure and composition of the component.

```tsx filename="Anatomy"
<Button>Label</Button>
```

## Examples

### Variants

The `variant` prop can be used to change the variant of the button.

<Preview name="variants" />
### Outline

Outline buttons are buttons that have a border.
This can be achieved by using the `variant="outline"` prop.

<Preview name="outline" />
### Plain

Plain button is a button that has no background color.
This can be achieved by using the `variant="plain"` prop.

<Preview name="plain" />
### Size

Buttons come in a variety of sizes. Use the `size` prop to change the size of the button.

<Preview name="size" />
### Block

Use the `block` prop to make the button fill the width of its container.

<Preview name="block" />
### Pill

Use the `pill` prop to create a pill-shaped button.

<Preview name="pill" />
### Icon

Buttons can have icons. Place the icon inside the button and it will be sized automatically.
You can also use the `size="icon"` prop to create an icon-only button.

<Preview name="icon" />
### Progress

Use the `progress` prop to show a loading indicator.

<Preview name="progress" />
### Icon with Progress

When using an icon with progress, the icon will be hidden during the loading state and the spinner will be shown instead.

<Preview name="iconProgress" />

### Link

You may want to use buttons as links.
This can be achieved by using the `render` prop to render a custom element.

<Preview name="link" />
### Disabled

Use the `disabled` prop to make the button disabled.

<Preview name="disabled" />
