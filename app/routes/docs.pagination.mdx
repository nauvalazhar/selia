# Pagination

Navigate through pages of content with numbered or directional controls.

<DocsButtons />

<Preview name="basic" />
## Usage

### Import

Import the component and its related parts.

```ts filename="Pagination Parts"
import {
  Pagination,
  PaginationList,
  PaginationItem,
  PaginationButton
} from "components/selia/pagination";
```

### Structure

The expected structure and composition of the component.

```tsx filename="Anatomy"
<Pagination>
  <PaginationList>
    <PaginationItem>
      <PaginationButton>1</PaginationButton>
    </PaginationItem>
    <PaginationItem>
      <PaginationButton>2</PaginationButton>
    </PaginationItem>
    <PaginationItem>
      <PaginationButton>3</PaginationButton>
    </PaginationItem>
    <PaginationItem>
      <PaginationButton>4</PaginationButton>
    </PaginationItem>
  </PaginationList>
</Pagination>
```
