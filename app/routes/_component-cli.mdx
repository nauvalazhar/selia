## Components

Use the Selia CLI to add components to your project:

<CodeTabs items={['npx', 'pnpx', 'yarn', 'bunx']}>
  <CodeTabsPanel value="npx" language="bash">
    npx selia@latest add button
  </CodeTabsPanel>
  <CodeTabsPanel value="pnpx" language="bash">
    pnpx selia@latest add button
  </CodeTabsPanel>
  <CodeTabsPanel value="yarn" language="bash">
    yarn selia@latest add button
  </CodeTabsPanel>
  <CodeTabsPanel value="bunx" language="bash">
    bunx selia@latest add button
  </CodeTabsPanel>
</CodeTabs>
