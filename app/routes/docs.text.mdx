## Usage

### Import

Import the component and its related parts.

```ts filename="Text Parts"
import {
  Text,
  TextLink,
  Strong,
  Code
} from "components/selia/text";
```

### Structure

The expected structure and composition of the component.

```tsx filename="Anatomy"
<Text>Text</Text>
<TextLink>TextLink</TextLink>
<Strong>Strong</Strong>
<Code>Code</Code>
```

You may want to use the `render` prop to pass a custom link element. See the [Customization](/docs/customization#render-prop) page for more details.

```tsx filename="Custom Element"
import { Link } from "react-router";

<TextLink render={<Link to="/about" />}>
  About
</TextLink>
```
