# Avatar

Display user profile pictures or initials in a circular or rounded format.

## Usage

### Import

Import the component and its related parts.

```ts filename="Avatar Parts"
import {
  Avatar,
  AvatarImage,
  AvatarFallback,
  AvatarIndicator
} from "components/selia/avatar";
```

### Structure

The expected structure and composition of the component.

```tsx filename="Anatomy"
<Avatar>
  <AvatarImage
    src="..."
    alt="Avatar"
  />
  <AvatarFallback>NA</AvatarFallback>
</Avatar>
```
