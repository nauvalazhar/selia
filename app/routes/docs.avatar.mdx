
# Avatar

Display user profile pictures or initials in a circular or rounded format.

<Preview name="basic" />
## Anatomy

Import the component and assemble its parts:

```tsx filename="Anatomy"
<Avatar>
  <AvatarImage />
  <AvatarFallback />
  <AvatarIndicator />
</Avatar>
```

## Usage

### Initials

Create an avatar with only initials.

<Preview name="initial" />
### Indicator

The `AvatarIndicator` component lets you add a small indicator to the avatar.
It can be used to indicate the presence of a new message, a notification, or a badge.
