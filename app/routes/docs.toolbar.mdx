# Toolbar

A container for grouping a set of buttons and controls.

<Preview name="basic" start />

## Anatomy

Import the component and assemble its parts:

```tsx filename="Anatomy"
<Toolbar>
  <ToolbarButton />
  <ToolbarLink />
  <ToolbarSeparator />
  <ToolbarGroup>
    <ToolbarButton />
  <ToolbarGroup />
  <ToolbarInput />
</Toolbar>
```
