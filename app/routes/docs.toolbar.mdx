## Usage

### Import

Import the component and its related parts.

```ts filename="Toolbar Parts"
import {
  Toolbar,
  ToolbarButton,
  ToolbarLink,
  ToolbarInput,
  ToolbarGroup,
  ToolbarSeparator
} from "components/selia/toolbar";
```

### Structure

The expected structure and composition of the component.

```tsx filename="Anatomy"
<Toolbar>
  <ToolbarButton>Button</ToolbarButton>
  <ToolbarLink>Link</ToolbarLink>
  <ToolbarSeparator />
  <ToolbarGroup>
    <ToolbarButton>Button</ToolbarButton>
  </ToolbarGroup>
  <ToolbarInput />
</Toolbar>
```
