## Usage

### Import

Import the component and its related parts.

```ts filename="Tooltip Parts"
import {
  Tooltip,
  TooltipTrigger,
  TooltipPopup
} from "components/selia/tooltip";
```

### Structure

The expected structure and composition of the component.

```tsx filename="Anatomy"
<Tooltip>
  <TooltipTrigger>Trigger</TooltipTrigger>
  <TooltipPopup>Content goes here</TooltipPopup>
</Tooltip>
```
