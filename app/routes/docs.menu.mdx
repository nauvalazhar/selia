# Menu

Reveal a list of actions or options when triggered by a button.

## Usage

### Import

Import the component and its related parts.

```ts filename="Menu Parts"
import {
  Menu,
  MenuTrigger,
  MenuPopup,
  MenuItem,
  MenuSeparator,
  MenuSubmenu,
  MenuSubmenuTrigger,
  MenuSubPopup,
  MenuGroup,
  MenuGroupLabel,
  MenuCheckboxItem,
  MenuRadioGroup,
  MenuRadioItem
} from "components/selia/menu";
```

### Structure

The expected structure and composition of the component.

```tsx filename="Anatomy" clip
<Menu>
  <MenuTrigger>Menu</MenuTrigger>
  <MenuPopup>
    <MenuGroup>
      <MenuGroupLabel>Account</MenuGroupLabel>
      <MenuItem>
        <UserIcon />
        Profile
      </MenuItem>
      <MenuItem>
        <Settings2Icon />
        Settings
      </MenuItem>
    </MenuGroup>
    <MenuSeparator />
    <MenuGroup>
      <MenuGroupLabel>Appearances</MenuGroupLabel>
      <MenuSubmenu>
        <MenuSubmenuTrigger>Theme</MenuSubmenuTrigger>
        <MenuPopup>
          <MenuItem>Light</MenuItem>
          <MenuItem>Dark</MenuItem>
          <MenuItem>System</MenuItem>
        </MenuPopup>
      </MenuSubmenu>
      <MenuItem>
        Toggle Sidebar
      </MenuItem>
    </MenuGroup>
    <MenuSeparator />
    <MenuItem>
      Quit App
    </MenuItem>
  </MenuPopup>
</Menu>
```

### Trigger

You can pass a custom trigger element to the `MenuTrigger` component
by using the `render` prop.

<Preview name="trigger" />
### Icon

You can add an icon directly to the menu item.

<Preview name="icon" />
### Nested

Menu items can be nested to create a hierarchy of options.

<Preview name="nested" />
### Group

You can group menu items by using the `MenuGroup` component and add a label to the group by using the `MenuGroupLabel` component.

<Preview name="group" />
### Radio

Let users select a single option from a set of choices with `MenuRadioGroup` component or multiple options with `MenuCheckboxItem` component.

<Preview name="radio" />
