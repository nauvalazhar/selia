# Combobox

Combined input and dropdown for selecting or entering values with autocomplete.

<Preview name="basic">
  <PreviewDemo>
    <examples.basic.component />
  </PreviewDemo>
</Preview>

## Anatomy

Import the component and assemble its parts:

```tsx filename="Anatomy"
<Combobox>
  <ComboboxInput />
  <ComboboxTrigger>
    <ComboboxValue />
  </ComboboxTrigger>
  <ComboboxContent>
    <ComboboxSearch />
    <ComboboxEmpty />
    <ComboboxList>
      <ComboboxItem />
    </ComboboxList>
  </ComboboxContent>
</Combobox>
```

## Usage

### Variants

You can choose between a `subtle` or `plain` variant.
