# Combobox

Combined input and menu for selecting or entering values with autocomplete.

<DocsButtons externalDoc="https://base-ui.com/react/components/combobox" />

<Preview name="basic" />
## Anatomy

Import the component and assemble its parts:

```tsx filename="Anatomy"
<Combobox>
  <ComboboxInput />
  <ComboboxTrigger>
    <ComboboxValue />
  </ComboboxTrigger>
  <ComboboxPopup>
    <ComboboxSearch />
    <ComboboxEmpty />
    <ComboboxList>
      <ComboboxItem />
    </ComboboxList>
  </ComboboxPopup>
</Combobox>
```

## Usage

### Variants

Combobox comes in a variety of variants, each with its own color and style.
You can choose between a `subtle` or `plain` variant.
