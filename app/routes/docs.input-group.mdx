## Usage

### Import

Import the component and its related parts.

```ts filename="Input Group Parts"
import {
  InputGroup,
  InputGroupAddon,
  InputGroupText
} from "components/selia/input-group";
```

### Structure

The expected structure and composition of the component.

```tsx filename="Anatomy"
<InputGroup>
  <InputGroupAddon>
    <InputGroupText />
  </InputGroupAddon>
</InputGroup>
```
