
# Input Group

Combine inputs with add-ons like buttons, icons, or text labels.

<Preview name="basic" />
## Anatomy

Import the component and assemble its parts:

```tsx filename="Anatomy"
<InputGroup>
  <InputGroupAddon>
    <InputGroupText />
  </InputGroupAddon>
  <InputGroupSeparator />
  <InputGroupBar />
</InputGroup>
```
