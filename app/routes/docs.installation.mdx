# Installation

Setup Selia for your project.

## Dependencies

First, install the dependencies:

```bash filename="Terminal"
pnpm add @base-ui-components/react tailwind-merge clsx class-variance-authority
```

## Helper Function

Create a new helper function to merge class names:

```ts filename="lib/utils.ts"
import { clsx, type ClassValue } from 'clsx';
import { twMerge } from 'tailwind-merge';

export function cn(...inputs: ClassValue[]) {
  return twMerge(clsx(inputs));
}
```

## Path Aliases

Add the `baseUrl` compiler option to your `tsconfig.json` file:

```json filename="tsconfig.json"
{
  "compilerOptions": {
    "baseUrl": ".",
  }
}
```

## Font Family (optional)

You can use any font you like, but Selia was designed with [Inter](https://fonts.google.com/specimen/Inter) and [JetBrains Mono](https://fonts.google.com/specimen/JetBrains+Mono).

Add the following to your CSS file:

```css filename="app.css"
@theme inline {
  --font-sans:
    'Inter', ui-sans-serif, system-ui, sans-serif, 'Apple Color Emoji',
    'Segoe UI Emoji', 'Segoe UI Symbol', 'Noto Color Emoji';
  --font-mono:
    'JetBrains Mono', ui-monospace, SFMono-Regular, Menlo, Monaco, Consolas,
    'Liberation Mono', 'Courier New', monospace;
}
```
