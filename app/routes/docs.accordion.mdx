# Accordion

Display content in a vertically stacked set of panels.

<Preview name="basic" />

## Anatomy

Import the component and assemble its parts:

```tsx filename="Anatomy"
<Accordion>
  <AccordionItem>
    <AccordionHeader>
      <AccordionTrigger />
    </AccordionHeader>
    <AccordionPanel />
  </AccordionItem>
</Accordion>
```
