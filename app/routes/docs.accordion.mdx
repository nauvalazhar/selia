## Usage

### Import

Import the component and its related parts.

```tsx filename="Accordion Parts"
import { 
  Accordion, 
  AccordionItem, 
  AccordionHeader, 
  AccordionTrigger, 
  AccordionPanel 
} from 'components/selia/accordion';
```

### Structure

The expected structure and composition of the component.

```tsx filename="Anatomy"
<Accordion>
  <AccordionItem>
    <AccordionHeader>
      <AccordionTrigger>
        Accordion Trigger
      </AccordionTrigger>
    </AccordionHeader>
    <AccordionPanel>
      Accordion content
    </AccordionPanel>
  </AccordionItem>
</Accordion>
```
