
# Field

Wrapper component for form inputs with label and validation support.

<Preview name="basic" />
## Anatomy

Import the component and assemble its parts:

```tsx filename="Anatomy"
<Field>
  <FieldLabel />
  <Input />
  <FieldError />
  <FieldDescription />
</Field>
```
