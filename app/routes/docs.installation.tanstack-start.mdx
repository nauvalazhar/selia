# Install Selia with TanStack Start

Setting up Selia in TanStack Start project.

<DocsButtons />

## Create Project

Create a TanStack Start project with Tailwind CSS:

<CodeTabs items={['npm', 'pnpm', 'yarn', 'bun']} id="pm">
  <CodeTabsPanel value="npm" language="bash">
    npm create @tanstack/start@latest --tailwind
  </CodeTabsPanel>
  <CodeTabsPanel value="yarn" language="bash">
    yarn create @tanstack/start@latest --tailwind
  </CodeTabsPanel>
  <CodeTabsPanel value="pnpm" language="bash">
    pnpm create @tanstack/start@latest --tailwind
  </CodeTabsPanel>
  <CodeTabsPanel value="bun" language="bash">
    bun create @tanstack/start@latest --tailwind
  </CodeTabsPanel>
</CodeTabs>

## Selia Initialization

Run the following command to init Selia:

<CodeTabs items={['npm', 'pnpm', 'yarn', 'bun']} id="pm">
  <CodeTabsPanel value="npm" language="bash">
    npx selia@latest init
  </CodeTabsPanel>
  <CodeTabsPanel value="pnpm" language="bash">
    pnpx selia@latest init
  </CodeTabsPanel>
  <CodeTabsPanel value="yarn" language="bash">
    yarn selia@latest init
  </CodeTabsPanel>
  <CodeTabsPanel value="bun" language="bash">
    bunx selia@latest init
  </CodeTabsPanel>
</CodeTabs>
