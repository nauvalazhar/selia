# Install Selia with Next.js

Setting up Selia in Next.js project.

<DocsButtons />

## Create Project

Create a Next.js project with recommended settings:

<CodeTabs items={['npm', 'pnpm', 'yarn', 'bun']} id="pm">
  <CodeTabsPanel value="npm" language="bash">
    npx create-next-app@latest my-app --yes
  </CodeTabsPanel>
  <CodeTabsPanel value="yarn" language="bash">
    yarn create next-app@latest my-app --yes
  </CodeTabsPanel>
  <CodeTabsPanel value="pnpm" language="bash">
    pnpm create next-app@latest my-app --yes
  </CodeTabsPanel>
  <CodeTabsPanel value="bun" language="bash">
    bun create next-app@latest my-app --yes
  </CodeTabsPanel>
</CodeTabs>

## Selia Initialization

Run the following command to init Selia:

<CodeTabs items={['npm', 'pnpm', 'yarn', 'bun']} id="pm">
  <CodeTabsPanel value="npm" language="bash">
    npx selia@latest init
  </CodeTabsPanel>
  <CodeTabsPanel value="pnpm" language="bash">
    pnpx selia@latest init
  </CodeTabsPanel>
  <CodeTabsPanel value="yarn" language="bash">
    yarn selia@latest init
  </CodeTabsPanel>
  <CodeTabsPanel value="bun" language="bash">
    bunx selia@latest init
  </CodeTabsPanel>
</CodeTabs>
