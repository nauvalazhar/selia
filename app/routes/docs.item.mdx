
# Item

Versatile component for combining media, text, and actions in flexible layouts.

<DocsButtons />

<Preview name="basic" />
## Guidelines

You can treat the `Item` component as a classic media object component. 
It is flexible and can be used in various ways.
Consider using it before creating a new component.

## Usage

### Import

Import the component and its related parts.

```ts filename="Item Parts"
import {
  Item,
  ItemContent,
  ItemTitle,
  ItemDescription,
  ItemMeta,
  ItemMedia,
  ItemAction
} from "components/selia/item";
```

### Structure

The expected structure and composition of the component.

```tsx filename="Anatomy"
<Item>
  <ItemMedia />
  <ItemContent>
    <ItemTitle />
    <ItemDescription />
    <ItemMeta />
  </ItemContent>
  <ItemAction />
</Item>
```
