# Card

Flexible container for grouping related content with optional headers and footers.

<Preview name="basic">
  <PreviewDemo>
    <examples.basic.component />
  </PreviewDemo>
</Preview>

## Anatomy

Import the component and assemble its parts:

```tsx filename="Anatomy"
<Card>
  <CardHeader>
    <CardTitle />
    <CardDescription />
    <CardHeaderAction />
  </CardHeader>
  <CardBody />
  <CardFooter />
</Card>
```
