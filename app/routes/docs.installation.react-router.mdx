# Install Selia with React Router

Setting up Selia in React Router project.

<DocsButtons />

## Create Project

Create a React Router project:

<CodeTabs items={['npm', 'pnpm', 'yarn', 'bun']}>
  <CodeTabsPanel value="npm" language="bash">
    npx create-eact-router@latest my-app
  </CodeTabsPanel>
  <CodeTabsPanel value="yarn" language="bash">
    yarn create react-router my-app
  </CodeTabsPanel>
  <CodeTabsPanel value="pnpm" language="bash">
    pnpm create react-router@latest my-app
  </CodeTabsPanel>
  <CodeTabsPanel value="bun" language="bash">
    bun create react-router@latest my-app
  </CodeTabsPanel>
</CodeTabs>

## Selia Initialization

Run the following command to init Selia:

<CodeTabs items={['npx', 'pnpx', 'yarn', 'bunx']}>
  <CodeTabsPanel value="npx" language="bash">
    npx selia@latest init
  </CodeTabsPanel>
  <CodeTabsPanel value="pnpx" language="bash">
    pnpx selia@latest init
  </CodeTabsPanel>
  <CodeTabsPanel value="yarn" language="bash">
    yarn selia@latest init
  </CodeTabsPanel>
  <CodeTabsPanel value="bunx" language="bash">
    bunx --bun selia@latest init
  </CodeTabsPanel>
</CodeTabs>
