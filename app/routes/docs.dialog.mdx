# Dialog

Modal windows that focus user attention on specific content or actions.

<DocsButtons externalDoc="https://base-ui.com/react/components/dialog" />

<Preview name="basic" />
## Anatomy

Import the component and assemble its parts:

```tsx filename="Anatomy"
<Dialog>
  <DialogTrigger />
  <DialogPopup>
    <DialogHeader>
      <DialogTitle />
    </DialogHeader>
    <DialogBody>
      <DialogDescription />
    </DialogBody>
    <DialogFooter>
      <DialogClose />
    </DialogFooter>
  </DialogPopup>
</Dialog>
```

## Usage

### Trigger

You can pass a custom trigger element to the `DialogTrigger` component
by using the `render` prop.
