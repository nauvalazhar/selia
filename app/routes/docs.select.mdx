
# Select

Dropdown menu for choosing single option from a list.

<Preview name="basic" />
## Anatomy

Import the component and assemble its parts:

```tsx filename="Anatomy"
<Select>
  <SelectTrigger>
    <SelectValue />
  </SelectTrigger>
  <SelectContent>
    <SelectList>
      <SelectItem />
    </SelectList>
  </SelectContent>
</Select>
```

## Usage

### Multiple

To allow users to select multiple options, pass the `multiple` prop to the `Select` component.

<Preview name="multiple" />
### Group

To group options, pass the `group` prop to the `Select` component.

<Preview name="group" />
### Icon

To add an icon to an option, you can place the icon inside the `SelectItem` component.

<Preview name="icon" />
### Disable

To disable an option, pass the `disabled` prop to the `SelectItem` component.

<Preview name="disable" />
