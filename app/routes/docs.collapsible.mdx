## Usage

### Import

Import the component and its related parts.

```ts filename="Collapsible Parts"
import {
  Collapsible,
  CollapsibleTrigger,
  CollapsiblePanel
} from "components/selia/collapsible";
```

### Structure

The expected structure and composition of the component.

```tsx filename="Anatomy"
<Collapsible>
  <CollapsibleTrigger>Trigger</CollapsibleTrigger>
  <CollapsiblePanel>Collapsible content</CollapsiblePanel>
</Collapsible>
```
