
# Popover

Display additional content in a floating overlay near a trigger element.

<Preview name="basic" />
## Anatomy

Import the component and assemble its parts:

```tsx filename="Anatomy"
<Popover>
  <PopoverTrigger />
  <PopoverContent>
    <PopoverTitle />
    <PopoverDescription />
  </PopoverContent>
</Popover>
```
