# Popover

Display additional content in a floating overlay near a trigger element.

<DocsButtons externalDoc="https://base-ui.com/react/components/popover" />

<Preview name="basic" />
## Usage

### Import

Import the component and its related parts.
```ts filename="Popover Parts"
import {
  Popover,
  PopoverTrigger,
  PopoverPopup,
  PopoverTitle,
  PopoverDescription
} from "components/selia/popover";
```

### Structure

The expected structure and composition of the component.

```tsx filename="Anatomy"
<Popover>
  <PopoverTrigger>Popover Trigger</PopoverTrigger>
  <PopoverPopup>
    <PopoverTitle>Popover Title</PopoverTitle>
    <PopoverDescription>Popover Description</PopoverDescription>
  </PopoverPopup>
</Popover>
```
