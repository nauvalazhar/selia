## Usage

### Import

Import the component and its related parts.

```tsx filename="Alert Parts"
import { 
  Alert, 
  AlertTitle, 
  AlertDescription, 
  AlertAction 
} from 'components/selia/alert';
```

### Structure

The expected structure and composition of the component.

```tsx filename="Anatomy"
<Alert>
  <AlertTitle>Alert Title</AlertTitle>
  <AlertDescription>
    Alert description goes here.
  </AlertDescription>
  <AlertAction>
    <Button>Action</Button>
  </AlertAction>
</Alert>
```

**Minimal Usage**

```tsx filename="Minimal Usage"
<Alert>
  Message goes here.
</Alert>
```
