# Alert

Component to display alerts.

## Anatomy

Import the component and assemble its parts:

```tsx filename="Anatomy"
<Alert>
  <AlertTitle />
  <AlertDescription />
</Alert>
```

### Variants

You may use different variants of the alert by using the `variant` prop.
