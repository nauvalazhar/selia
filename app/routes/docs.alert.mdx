## Anatomy

Import the component and assemble its parts:

```tsx filename="Anatomy"
import { 
  Alert, 
  AlertTitle, 
  AlertDescription 
} from 'components/selia/alert';

<Alert>
  <AlertTitle />
  <AlertDescription />
</Alert>
```
