# Preview Card

A popup that appears when a link is hovered, showing a preview for sighted users.

## Usage

### Import

Import the component and its related parts.

```ts filename="Preview Card Parts"
import {
  PreviewCard,
  PreviewCardTrigger,
  PreviewCardPopup
} from "components/selia/preview-card";
```

### Structure

The expected structure and composition of the component.

```tsx filename="Anatomy"
<PreviewCard>
  <PreviewCardTrigger>Trigger</PreviewCardTrigger>
  <PreviewCardPopup>
    Content goes here
  </PreviewCardPopup>
</PreviewCard>
```
