
# Table

Organize and display data in rows and columns in a tabular format.

<Preview name="basic" />
## Anatomy

Import the component and assemble its parts:

```tsx filename="Anatomy"
<TableContainer>
  <Table>
    <TableCaption />
    <TableHeader>
      <TableRow>
        <TableHead />
      </TableRow>
    </TableHeader>
    <TableBody>
      <TableRow>
        <TableCell />
      </TableRow>
    </TableBody>
    <TableFooter>
      <TableRow>
        <TableCell />
      </TableRow>
    </TableFooter>
  </Table>
</TableContainer>
```

## Usage

### Card

You can wrap the `Table` component in a `Card` component to create a card table.
