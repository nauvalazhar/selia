# Table

Organize and display data in rows and columns in a tabular format.

<Preview name="basic">
  <PreviewDemo>
    <examples.basic.component />
  </PreviewDemo>
</Preview>

## Anatomy

Import the component and assemble its parts:

```tsx filename="Anatomy"
<TableContainer>
  <Table>
    <TableCaption />
    <TableHeader>
      <TableRow>
        <TableHead />
      </TableRow>
    </TableHeader>
    <TableBody>
      <TableRow>
        <TableCell />
      </TableRow>
    </TableBody>
    <TableFooter>
      <TableRow>
        <TableCell />
      </TableRow>
    </TableFooter>
  </Table>
</TableContainer>
```
