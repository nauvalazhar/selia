## Usage

### Import

Import the component and its related parts.

```ts filename="checkbox Parts"
import {
  CheckboxGroup,
  CheckboxGroupLabel,
  Checkbox
} from "components/selia/checkbox";
```

### Structure

The expected structure and composition of the component.

```tsx filename="Anatomy"
<CheckboxGroup>
  <CheckboxGroupLabel>Group Label</CheckboxGroupLabel>
  <Checkbox />
</CheckboxGroup>
```

**Minimal Usage**

```tsx filename="Minimal Usage"
<Checkbox />
```
