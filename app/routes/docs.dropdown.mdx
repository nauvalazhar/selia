# Dropdown

Reveal a list of actions or options when triggered by a button.

<Preview name="basic">
  <PreviewDemo>
    <examples.basic.component />
  </PreviewDemo>
</Preview>

## Anatomy

Import the component and assemble its parts:

```tsx filename="Anatomy"
<Dropdown>
  <DropdownTrigger />
  <DropdownContent>
    <DropdownItem />
    <DropdownSeparator />
    <DropdownGroup>
      <DropdownGroupLabel />
    </DropdownGroup>
    <DropdownRadioGroup>
      <DropdownRadioItem />
    </DropdownRadioGroup>
    <DropdownCheckboxItem />
    <DropdownSubmenu>
      <DropdownSubmenuTrigger />
    </DropdownSubmenu>
  </DropdownContent>
</Dropdown>
```

## Usage

### Trigger

You can pass a custom trigger element to the `DropdownTrigger` component
by using the `render` prop.
