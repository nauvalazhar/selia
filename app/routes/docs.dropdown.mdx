
# Dropdown

Reveal a list of actions or options when triggered by a button.

<Preview name="basic" />
## Anatomy

Import the component and assemble its parts:

```tsx filename="Anatomy"
<Dropdown>
  <DropdownTrigger />
  <DropdownContent>
    <DropdownItem />
    <DropdownSeparator />
    <DropdownGroup>
      <DropdownGroupLabel />
    </DropdownGroup>
    <DropdownRadioGroup>
      <DropdownRadioItem />
    </DropdownRadioGroup>
    <DropdownCheckboxItem />
    <DropdownSubmenu>
      <DropdownSubmenuTrigger />
    </DropdownSubmenu>
  </DropdownContent>
</Dropdown>
```

## Usage

### Trigger

You can pass a custom trigger element to the `DropdownTrigger` component
by using the `render` prop.

<Preview name="trigger" />
### Icon

You can add an icon directly to the dropdown item.

<Preview name="icon" />
### Nested

Dropdown items can be nested to create a hierarchy of options.

<Preview name="nested" />
### Group

You can group dropdown items by using the `DropdownGroup` component and add a label to the group by using the `DropdownGroupLabel` component.

<Preview name="group" />
### Radio

Let users select a single option from a set of choices with `DropdownRadioGroup` component or multiple options with `DropdownCheckboxItem` component.

<Preview name="radio" />
